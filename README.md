 The Expense Tracker App is a modern Android application designed to help users effectively manage their finances by tracking daily expenses and providing insightful visualizations.
• Built with Jetpack Compose for a clean and interactive UI, the app features predictive analytics to forecast future expenses based on user behavior and historical data.
• With seamless data storage using Room Database and real-time updates via ViewModel, the app ensures data persistence and an intuitive user experience.
• Technological Stack: Jetpack Compose, Retrofit, REST APIs, Kotlin, Room Database, and ViewModel.
